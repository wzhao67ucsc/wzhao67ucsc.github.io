Floot (YC S25)

As a Founding Member, I led UI/UX improvements based on data-driven insights. Focused on optimizing the hero section, simplifying authentication flows, and redesigning paywalls with copywriting tailored to user intent.

Role

Founding Member / Designer

Platform

Desktop / Web

Fig 01. The redesigned hero section optimized for conversion based on PostHog analytics insights.

The Challenge

Using PostHog analytics, I identified key friction points in the user journey. The hero section wasn't converting effectively, the authentication flow was unnecessarily complex, and paywalls weren't resonating with user intent. The goal was to improve conversion rates through targeted UI/UX improvements backed by data.

My Work

Data-Driven Analysis

Analyzed user behavior and conversion funnels using PostHog to identify key friction points and opportunities for improvement across the platform.

Hero Section Redesign

Redesigned the hero section based on analytics data to improve clarity, messaging, and conversion rates for new users landing on the platform.

Simplified Authentication

Streamlined the authentication flow, reducing complexity and friction to create a smoother onboarding experience for users.

Intent-Based Paywalls

Redesigned paywall interfaces with copywriting tailored to user intent, improving conversion rates by addressing what users actually need at each stage of their journey.
